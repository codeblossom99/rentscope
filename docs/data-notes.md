# 資料源觀察筆記 — 內政部租賃實價登錄

樣本: 115S1 + 115S2 雙北租賃檔 (A/F_lvr_land_C.csv), 共 30,947 筆

## 檔案結構

- 每季一包, 每縣市 4 個檔: 主檔 `*_C.csv` + 附表 `*_C_build/land/park.csv`, 以 `編號` join
- 編碼 UTF-8 with BOM (`utf-8-sig`)
- **第 2 列是英文欄名**, parse 時要跳過
- 主檔欄位與 `schema-main.csv` (買賣用) 不同, 租賃有自己的欄位: 出租型態、有無附傢俱、租賃期間、有無電梯、租賃住宅服務

## 品質量測 (30,947 筆)

| 檢查項 | 結果 |
|---|---|
| `編號` 檔內重複 | 0, 可當 unique key |
| `編號` 跨季重複 | 1 筆 → upsert 需以 `編號` 冪等 |
| `租賃年月日` 格式異常 | 0, 全為 7 碼民國年 (如 `1141111`) |
| `總額元` 為 0 或空 | 0 |
| `建物總面積` 為 0 | 17 筆 (車位、土地案件) |
| `建築完成年月` 缺值 | ~5% |
| `出租型態` 缺值 | ~10% |
| `單價元平方公尺` 缺值 | <1% |

## 清洗規則 (ETL 需處理)

1. **民國年轉換**: `1141111` → `2025-11-11` (+1911); `建築完成年月` 有前導零 7 碼如 `0851105`
2. **全形字元**: 地址含全形數字 (`５８１ 巷`) → NFKC normalize
3. **交易標的過濾**: 含純 `車位` / `土地` 案件 (~0.4%), MVP 只留租賃房屋
4. **極端值**: 月租金 max 達 18,125,994 (整棟商用), 查詢預設應排 outlier 或依 `建物型態` 分流; 分布參考: p25=16k, 中位=24k, p75=38k (台北 115S2)
5. **租賃期間**: 格式 `1150101~1151231`, 拆 start/end
6. **備註**: 自由文字, 含「含管理費」「續租案件」等訊號, 先原樣保留於 raw JSONB
7. **季檔內含跨期資料**: 115S1 檔內有 114 年的租賃日 → 以 `租賃年月日` 為準, 不能拿季別當日期
